# blurText

> A lightweight JavaScript library for blurring text

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save blurtext
```

## Usage

```js
const { blurText } = require("blurtext");

console.log(blurText("beta", "#"))
```
